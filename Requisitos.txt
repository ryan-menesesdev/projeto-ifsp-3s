Meta: Implementação de interface de usuário de um sistema web (front-end).  

Data da apresentação: 15/01/2025 Auditório BANCA: Bianca, Eliana e Everton

Requisitos:

Implementar um site para o demandante  nas linguagens estudadas em Desenvolvimento WEB 1,  que são:htm, CSS, Javascript.
Hospedar o site num provedor de acesso
Entregar: links para repositório(git) contendo os scripts e da hospedagem do site.
O site tem que ter uma página principal(index.html), uma página sobre a equipe de desenvolvimento , uma página sobre(sobre.html) a demandante, uma página para as pessoas entrarem em contato, ou fornecer dados da empresa como whatsapp, telefone, endereço(faleconosco.html), outras páginas sobre produtos e serviços oferidos pelo demandante.
Critérios de avaliação:

- Acessibilidade/Tags semânticas

- Responsividade( contêineres e grid)

- Estilo (cores, alinhamento, imagens)

- Estrutura da página (organização dos itens em listas, tabelas, etc)

- Validação (w3validator)

- Navegação (nav, pagination, etc)
